# SBUS2-Telemetry
Arduino Library for Futaba SBUS2 Telemetry based on Atmega 328P

## Hardware

Tested with Arduino Pro Mini with 8MHz
Tested with Futaba T14SG
Tested with R7008


## Arduino IDE

Tested with Version 1.8.5
